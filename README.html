Application Overview

Sight & Sound is a web application that demonstrates how to integrate Azure AI services to perform advanced image analysis and convert the resulting descriptions into spoken audio. This guide will walk you through the setup process.

Flexible Image Input
Deep Image Analysis
Dynamic Text-to-Speech

Prerequisites Checklist

Before you begin, ensure you have the following tools installed and set up.

Setup & Deployment Steps

Create Azure Resources

1. Azure AI Services Account

In the Azure Portal, create a new **Azure AI services multi-service account**. After creation, navigate to its "Keys and Endpoint" page and copy the **Endpoint URL** and one of the **Keys**.

Create AI Services Resource

2. Azure Function App

Create a new **Function App**. Ensure you select **Node.js** as the runtime stack and choose the same region as your AI Services account for optimal performance.

Create Function App

Configure Environment Variables

In your Function App's "Configuration" settings, add the following application settings. This is how your code securely accesses your API keys and endpoint.

VISION_ENDPOINT
VISION_KEY
SPEECH_REGION
SPEECH_KEY

Deploy the Code

Clone the repository and use the Azure Functions Core Tools to publish the application. Enter your Function App's name below to generate the exact command.

func azure functionapp publish <YOUR_FUNCTION_APP_NAME>

Use the App

Once deployed, your application will be available at your Function App's URL. You can now start analyzing images!

Example URL: https://<YOUR_APP_NAME>.azurewebsites.net
